# Ashley AI Studio

**A Hybrid AI Platform combining legacy AI models (Deep Blue, ELIZA, Deep Thought) with modern language models, featuring advanced text editing, diffusion-based rendering, 3D modeling, and intelligent voice interaction.**

## Features

### 🧠 Core Editor
- Notepad-like interface with create, save, load functionality
- Real-time word and phrase prediction
- Letter prediction powered by neural networks
- Virtual keyboard with AI-enhanced commands
- Spelling, grammar, and semantic analysis

### 🤖 Hybrid AI Engine
- **Legacy Models**: Deep Blue chess engine logic, ELIZA chatbot patterns, Deep Thought heuristics
- **Modern ML**: Transformers, Q-learning, neural networks
- **Prediction**: Keystroke pattern recognition, text generation, probability-based suggestions
- **Search**: Vector-based semantic search, training on embeddings

### 🎨 Diffusion & 3D Rendering
- Physics-based diffusion (water ripples, wave propagation)
- Click and drag to create waves and fluid dynamics
- Energy exchange and momentum simulation
- 2D to 3D diffusion modeling without traditional meshes
- Real-time diffusion manipulation and warping
- 2D image-to-3D model conversion

### 🎮 Game Engine
- DirectX12 and OpenGL support
- Physics engine (Newton's laws, gravity, thermodynamics)
- Scene management with camera controls
- GUI/UI framework with 2D and 3D windows
- Skeleton/bone systems for animation
- Procedural terrain and texture generation

### 🔊 Media & Voice
- Text-to-speech (default voice: Zira)
- Speech-to-text (STT)
- Audio recording and playback
- Video playback and processing
- Voice model training from audio files
- Real-time messaging with audio support

### 💾 Database & Knowledge Management
- File management with shortcuts and database indexing
- Dictionary folder with word meanings and usage examples
- English grammar rules and references
- Compressed file detection (ZIP, RAR) and extraction
- Vector store for embeddings and semantic search
- Training dataset organization and management

### 💬 Ashley AI Agent
- Intelligent chatbot with real-time messaging
- Code generation and script creation
- Model improvement and testing
- Error detection and code fixing
- Backup and fallback systems
- Multi-agent collaboration

## Quick Start

```bash
# Clone repository
git clone https://github.com/flamebomb88/ashley-ai-studio.git
cd ashley-ai-studio

# Run launcher
python launcher.py
```

## Installation

### Prerequisites
- Python 3.10+
- pip/conda

### Setup

```bash
# Create virtual environment
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt

# Run application
python launcher.py
```
